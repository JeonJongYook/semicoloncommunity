# IntecSemiColon Sns Community

## 프로젝를 기초 설정하고 싶다면 아래 명령어를 입력하세요
```
npm install
```

### 컴파일 후 개발 서버를 실행 하고 싶다면 아래 명령어를 입력하세요
```
npm run serve
```

### 컴파일하고 싶다면 아래 명령어를 입력하세요
```
npm run build
```

### Lint하거나 파일을 수정하고 싶다면 아래 명령어를 입력하세요
```
npm run lint
```
